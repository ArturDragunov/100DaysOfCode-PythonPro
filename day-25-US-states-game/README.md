## Project Overview

An interactive educational game that challenges players to name and locate all 50 U.S. states on a blank map. Players learn state names and their geographical positions through an engaging, interactive experience.

## Key Features
- Graphical map of the United States using Turtle graphics
- Interactive state guessing mechanism
- Real-time tracking of correctly guessed states
- Option to exit and save unguessed states for later learning
- Displays states on the map as they are correctly identified
- Provides a CSV file of missed states for further study

## How to Play
1. Run `main.py` to start the game
2. Type the name of a U.S. state when prompted
3. Correctly guessed states will appear on the map
4. Track your progress with the score display (X/50 States Correct)
5. Type "Exit" to end the game and generate a list of states to learn
6. A `states_to_learn.csv` will be created with states you didn't guess
